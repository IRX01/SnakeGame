# Console Snake Game

Учебная реализация классической игры «Змейка» в консоли на C#.

## Функционал
* Управление змейкой с помощью клавиш WASD.
* Увеличение длины змейки при поедании «фрукта».
* Игра заканчивается при столкновении со стеной или собственным телом.
* Отображение координат игрока и фрукта.

## Технологии
* C#
* Console Application (.NET)

## Запуск
1. Откройте проект в Visual Studio / Rider.
2. Запустите `Program.cs`.
3. Используйте клавиши WASD для управления.
